One-time setup
Before you
get started
YourSpace needs a free API key to fetch videos from YouTube. This takes about 3 minutes to set up and you'll never need to touch it again.
What is an API key?

Think of it like a personal library card. It proves to YouTube that you're allowed to request their data, and it tracks your personal daily usage — not ours. Google gives every account 10,000 free requests per day, which is far more than you'll ever use just watching videos.

YourSpace is completely free. The API key is also free. Neither of us ever sees your key — it lives only in your browser on your device.

⚠️
Treat your API key like a password
  • Never share it with anyone
  • Never paste it into another app or website
  • Never send it in a message or email
  • If you think it's been exposed, delete it in Google Cloud and create a new one
1
Go to Google Cloud Console
console.cloud.google.com — sign in with your Google account. It's free, no payment needed.
2
Create a project
Click "New Project" at the top → give it any name (e.g. "YourSpace") → click Create. A project is just a folder that keeps your key organised.
3
Enable YouTube Data API v3
Go to APIs & Services → Library → search "YouTube Data API v3" → click it → click Enable. This gives your key permission to ask YouTube for video information.
4
Create your API key
Go to APIs & Services → Credentials → click "Create Credentials" → choose "API key". Google will show you a long string of letters and numbers — copy it.
5
Paste it below and save
Your key is saved in your browser only. It never leaves your device and is never shared with YourSpace or anyone else.

Only What You Choose

YourSpace is your personal, intentional digital environment — no algorithm, no recommendations, no infinite scroll. Just the channels you chose, the videos you actually want to watch.

Privacy

YourSpace does not collect, store, or transmit any personal data.

Your API key and channel preferences are stored locally in your browser only and never leave your device. We do not use cookies, analytics, or tracking of any kind.

Your API key communicates directly with YouTube's servers on your behalf — YourSpace never sees or handles this traffic.
