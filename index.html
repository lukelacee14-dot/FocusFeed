FocusFeed uses YouTube's free API to fetch videos — no algorithm, no tracking. You'll need a free API key. It takes about 3 minutes and you'll never need to touch it again.
1
Go to Google Cloud Console
console.cloud.google.com → sign in with your Google account
2
Create a project
Click "New Project" → name it anything (e.g. "FocusFeed") → Create
3
Enable YouTube Data API v3
APIs & Services → Library → search "YouTube Data API v3" → Enable
4
Create an API key
APIs & Services → Credentials → Create Credentials → API key → Copy it
5
Paste it below
